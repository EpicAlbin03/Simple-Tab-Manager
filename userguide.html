General Usage & Behavior

Please note that all settings are global, so any changes you save will be applied to all open windows. Similarly, the selected sorting options will also be saved and automatically applied when opening the extension again.

Select All

Selects all tabs. If all tabs are selected then unselect all of them.

Copy Urls

Copies the urls of the selected tabs to clipboard in form of a list.

Grouping options

Add to new group

Selected tabs will be added to a new group. Name and color is optional.

Add to existing group

Specify the name of the group and the selected tabs will be added to that group.

Remove from group

Specify the name of the group and the selected tabs will be removed from that group.

Ungroup

Specify the name of the group and all groups with that name will be ungrouped. The groups will be removed but the tabs will stay intact.

Close group

Specify the name of the group and all groups with that name will be closed. The groups, along with the tabs, will be removed.

Add to new window

Selected tabs will be added to a new window. Name is unavailable.

Sorting options

Sort by title

Sorts all tabs in the current window alphabetically by their titles. Tabs within groups will be unaffected if "Preserve order within groups" option is selected.

Sort by url

Sorts all tabs in the current window alphabetically by their urls. The tabs will be sorted by their hostname excluding things like "www" (see example below). Tabs within groups will be unaffected if "Preserve order within groups" option is selected.

Settings

Automatically sort tabs

All tabs in the current window will be automatically sorted after the currently selected sorting option (title or url) after [x] amount of time. For example if sec = 30, the tabs will be sorted every 30 seconds. Please note that at least one of "sec", "min" or "hours" has to be > 0.

Preserve order within groups

If this option is selected, groups will be unaffected by sorting. All groups will be moved to the left most positions and be sorted in alphabetical order. All tabs within groups will remain in the same order while the rest of the tabs will be sorted normally. If this option is unselected, groups won't be accounted for when sorting and will most likely ungroup tabs.
